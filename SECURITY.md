# Security

## Reporting a vulnerability

If you believe you've found a security issue, please report it responsibly:

- **Do not** open a public issue for security vulnerabilities.
- Email the maintainer (see GitHub profile or repo description) with a description of the issue and steps to reproduce, if possible.
- We will respond and work with you on a fix and disclosure timeline.

This app stores data only in the browser (localStorage). It does not send data to a server by default. If you deploy it with a backend or analytics, ensure you follow secure practices (HTTPS, no secrets in client code, etc.).
